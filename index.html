Run the Demo Locally

1. Clone & Install

git clone https://github.com/NullLabTests/math-reasoning-agent.git
cd math-reasoning-agent
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

2. Launch the Web UI

python app.py

Opens at http://localhost:7860

3. Run Evaluation

# Score baseline model
python scripts/baseline_eval.py --model PrimeIntellect/Qwen3-0.6B-Base --problems 20

# Compare baseline vs trained model
python scripts/post_training_eval.py \
    --baseline PrimeIntellect/Qwen3-0.6B-Base \
    --trained ./checkpoints/step-100 \
    --problems 20

How RL Training Works

  1. Environment loads math problems (AIME, GSM8K, or custom) and defines reward functions
  2. vLLM serves the base model and generates completion rollouts
  3. Orchestrator sends prompts → collects completions → scores them via the environment
  4. Trainer uses GRPO to update model weights, maximizing reward
  5. Repeat — model improves math reasoning over 50-200 steps
